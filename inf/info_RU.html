Как программировать одновременно на Java, Kotlin и Clojure
Интеграция Java, Kotlin и Clojure в одном программном продукте
Сравнительный анализ программирование на Java, Kotlin и Clojure 2023.01.01

- В статье и примерах проводится сравнительный анализ программирования на языках Java, Kotlin и Clojure с использованием среды программирования Eclipse и системы сборки Gradle.
- Разрабатываются конкретные примеры программ на Java, Kotlin и Clojure для создания основы для различных приложений с графическим интерфейсом.
- Программирование проводится одинаковым способом на всех языках программирования для возможности сравнения элементов программ.
- Примеры предназначены как основа для создания программ с графическим интерфейсом с возможностью его расширения для любых прикладных приложений.
- Расширение разработанных программ будет продолжено в направлении создания многомодульной системы с неизменной головной частью и с возможностью загрузки в нее функциональных модулей из депозитария, написанных как на Java, так и на Kotlin или Clojure. Предполагается открытый тип депозитария для возможности любого разработчика представлять в нем свои модули для демонстрации или использования.
- Прорабатывается возможность дополнения данной разработки языками программирования Scala (Java) и Python.
